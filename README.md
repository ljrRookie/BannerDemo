# Android图片轮播控件
Android广告图片轮播控件，支持无限循环和多种主题，可以灵活设置轮播样式、动画、轮播和切换时间、位置、图片加载框架等！

## Banner
实现了图片轮播效果，支持无限循环和多种主题，可以灵活设置轮播样式、动画、轮播和切换时间、位置、图片加载框架等！<br>
添加依赖：compile 'com.youth.banner:banner:1.4.2'<br>
